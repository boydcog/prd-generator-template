## **6\. AI 생성 지침**

### **6-1. 기술 스택 지침**

```
기술 스택:
- Framework: Next.js 14 (App Router)
- UI: shadcn/ui + Tailwind CSS
- 아이콘: lucide-react
- 폼: react-hook-form + zod
- 상태: [React Context / Zustand]
- 이미지: next/image (WebP, lazy loading)

컴포넌트 규칙:
- shadcn/ui 컴포넌트를 최우선으로 사용한다.
- className에는 Tailwind 유틸리티 클래스만 사용한다.
- CSS 파일을 새로 만들지 않는다.
- 컴포넌트는 app/components/ 또는 components/ui/에 배치한다.

접근성:
- 텍스트 없는 버튼에는 aria-label 필수.
- 이미지에 alt 속성 필수.
- 색상 대비 WCAG AA(4.5:1) 이상.
```

### **6-2. 비주얼 시스템 지침**

```
디자인 토큰 (globals.css CSS 변수 사용):
- Primary: [색상값] → bg-primary, text-primary
- Background: [색상값] → bg-background
- Foreground: [색상값] → text-foreground
- Success: [색상값] → text-green-600
- Warning: [색상값] → text-amber-500

타이포그래피:
- 본문: font-sans text-base text-foreground
- 제목: font-bold text-2xl tracking-tight
- 보조: text-sm text-muted-foreground

레이아웃:
- 페이지 패딩: px-4 (모바일) / px-6 (sm 이상)
- 컨테이너: max-w-[값] mx-auto
- 섹션 간격: space-y-6

이미지:
- next/image 컴포넌트 사용. width, height 또는 fill 속성 지정.
- 에셋 소스: [CDN URL 패턴]
```
