## **2\. 문제 정의**

| 항목 | 내용 |
| ----- | ----- |
| **타겟 고객** | 수학 성취도 하위 20% 아동(만4세\~초2)의 학부모. 6개월 이상 사교육을 시켰으나 기초 연산이 해결되지 않아 "밑 빠진 독에 물 붓기"를 체감한 부모. 연 600만원 사교육비 지출 중 \[데이터: 2024 사교육비 통계\] |
| **핵심 문제** | 아이가 "114-17"을 103이라 답하고, 어제 배운 연산 규칙을 오늘 완전히 잊는 '수학 기억상실증(Math Amnesia)'. 원인은 노력 부족이 아니라 **수감각(직산, 수 세기, 자릿값) 결핍**이라는 뇌신경학적 문제. 기존 사교육은 이 근본 원인에 개입하지 못한다 |
| **문제의 빈도/강도** | 매일 2시간 학습 전쟁. 부모와 아이 모두 번아웃. 초3 수학 미달 25% → 방치 시 9학년 포기율 49%로 급증 \[데이터: 교육부 통계\]. 학부모 커뮤니티에서 "devastated", "눈물 펑펑", "미칠 노릇" 등 극단적 감정 표현 다수 \[데이터: Mumsnet, 82cook, Reddit 20건\] |
| **현재 대안** | 수학학원(월 25~~40만원), 전문 치료(회당 9.6~~12.8만원), 학습앱(아이설렘 월 3.9만원, 토도수학 월 1.2만원), 학습지 |
| **현재 대안의 한계** | 학원: 단순 반복(양치기) — 수감각 결핍이라는 근본 원인에 개입 불가. 치료: 고비용+접근성 한계. 기존 앱: 고정 커리큘럼, 실시간 난이도 조절 없음, 오답 원인 분석 부재 \[데이터: 경쟁사 리뷰 분석\] |

## **4\. 솔루션 가설**

| 항목 | 내용 |
| ----- | ----- |
| **핵심 가치 제안** | AI가 아동의 오답 원인(수감각 결핍)을 실시간 분석하여, 막힌 지점을 잘게 쪼갠 하위 과제로 되돌아가 훈련시킨다 |
| **차별점** | ① **원인 기반 역동적 난이도 조절** — 비슷한 문제 반복이 아닌 오답 원인 분해 ② **반응시간(유창성)+오답패턴 분석** — 정오답만이 아닌 "왜 틀렸는지" 진단 ③ **공감형 피드백** — "틀렸어요" 대신 "이건 조금 어려웠죠", 실패를 발견 언어로 재구성 |
| **핵심 가정** | ① 수감각 훈련이 연산 정확도와 유창성을 개선한다 (효과) ② 막히는 지점을 잘게 쪼갠 하위 과제를 즉각 제시하면 아동이 혼자 해결 가능하다 (자기주도) ③ 공감형 피드백이 학습 지속 시간과 재방문율을 높인다 (리텐션) |
