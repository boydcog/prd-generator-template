> 역할: marketing | 프로젝트: 수학 문제 레벨링 품질 검증 | 문서 유형: PRD

## 1. 포지셔닝 전략

### 핵심 포지션

**"인지과학 기반 수학 난이도 측정의 표준"**

기존 에듀테크 솔루션은 문제 난이도를 주관적 판단(교사 경험)이나 단순 정답률 기반으로 분류한다. 본 제품은 6가지 인지 부하 가중치(W1~W6)를 정량적으로 조합하여 W(1~23) 단계와 C(1~15) 커리큘럼 레벨로 변환하는 **객관적이고 재현 가능한 난이도 측정 체계**를 제공한다. [CLM-001, CLM-002]

### 시장 내 위치

| 축 | 기존 솔루션 | 본 제품 |
|---|---|---|
| 난이도 기준 | 주관적/정답률 기반 | 6가지 인지 부하 가중치 정량화 |
| 문제 세분화 | 상/중/하 3단계 | W 23단계 + C 15레벨 이중 체계 |
| 적응형 근거 | 블랙박스 알고리즘 | 투명한 가중치 공식 (검증 가능) |
| 커버리지 | 전 과목 얕은 커버 | 덧셈/뺄셈 15,000문제 완전 커버 |

### 포지셔닝 맵

- **X축**: 난이도 분류 정밀도 (낮음 → 높음)
- **Y축**: 과목 커버리지 범위 (좁음 → 넓음)
- **본 제품 위치**: 높은 정밀도 + 좁은 커버리지 (덧셈/뺄셈 전문)
- **확장 방향**: 정밀도 유지하면서 곱셈/나눗셈으로 커버리지 확대

## 2. 핵심 메시징 필러

### 필러 1: 과학적 정밀도

> "감이 아닌, 과학으로 측정하는 수학 난이도"

- 6가지 가중치(연산 종류, 자릿수, 숫자 크기, 절차적 복잡성, 특수 숫자, 표현 방식)가 인지과학 원리에 기반 [CLM-001]
- carry/borrow 여부가 최대 +5.0점으로 난이도에 가장 큰 영향을 미침 -- 학생이 실제로 어려워하는 지점을 정확히 포착 [CLM-003]
- 10의 배수와 같은 심리적 안착점에 감점(-1.0점)을 부여하여 인지 부하의 미세한 차이까지 반영 [CLM-004]

### 필러 2: 세밀한 단계 설계

> "23단계로 나눈 학습 경로, 15레벨로 매핑한 커리큘럼"

- W(1~23) 단계: 0+0(W=1)부터 99+99 연속 carry(W=23)까지 23개 난이도 구간 [CLM-006]
- C(1~15) 레벨: 교과과정 진도에 맞춘 커리큘럼 매핑 [CLM-002]
- 약 15,000개 문제가 각각 고유한 난이도 점수를 보유하여 미세 조절 가능 [CLM-005, CLM-007]

### 필러 3: 투명한 기준

> "왜 이 문제가 어려운지, 누구나 이해할 수 있습니다"

- 각 문제의 난이도 점수가 6가지 가중치의 합산으로 설명 가능 [CLM-008]
- 블랙박스가 아닌 오픈 공식: 교사와 학부모가 난이도 판정 근거를 직접 확인 가능
- B2B 파트너에게 API로 가중치별 점수를 제공하여 자체 커스터마이징 지원

### 필러 4: 맞춤 학습 설계 기반

> "학생 수준에 딱 맞는 다음 문제를 자동으로"

- carry 없는 덧셈(W=6~8) → 1회 carry(W=15) → 연속 carry(W=23)로 이어지는 명확한 학습 경로 [CLM-005, CLM-006]
- 중상위 구간(W=18, C=11)은 초등 중학년 핵심 학습 영역으로 시장 수요 최대 [CLM-007]
- 학생이 현재 위치한 W단계에서 +1~2단계 문제를 자동 추천하는 적응형 학습 엔진 구현 가능

## 3. 타겟 채널

| 에듀테크 컨퍼런스 | EdTech Korea, BETT Asia 등 발표/부스 | 상 |
| 개발자 커뮤니티 | GitHub 오픈소스 데모, 기술 문서 공개 | 상 |
| 직접 영업 | 주요 에듀테크 사업자(매쓰홀릭, 클래스팅 등) 대상 PoC 제안 | 상 |
| 교육 관련 학회 | 한국수학교육학회 등 논문/발표를 통한 학술적 신뢰 확보 | 중 |

### 2차 타겟: B2B2C (학원/학교)

| 채널 | 전략 | 우선순위 |
|------|------|---------|
| 학원 업계 매체 | 학원관리 매거진, 학원연합신문 등 사례 기사 | 상 |

### 3차 타겟: B2C (학부모/학생)

| 채널 | 전략 | 우선순위 |

## 4. 런칭 단계 계획

### Phase 0: 사전 검증 (런칭 전 3개월)

- 가중치 체계의 교육학적 타당성 검증 (교육학 전문가 자문, 파일럿 테스트)
- 핵심 에듀테크 파트너 3~5곳 PoC 진행
- 기술 백서 작성 및 배포

### Phase 1: 클로즈드 베타 (런칭 전 2개월)

- B2B 파트너사 API 연동 테스트
- 파일럿 학원/학교 5~10곳에서 실사용 데이터 수집
- 사용자 피드백 기반 W/C 매핑 정합성 검증
- 성과 메트릭: API 호출 안정성 99.5%, 교사 만족도 4.0/5.0 이상

### Phase 2: 퍼블릭 런칭 (D-Day)

- **핵심 메시지**: "15,000개 수학 문제, 23단계 난이도 과학으로 분류"
- B2B API 공식 오픈 + 기술 문서 공개
- 에듀테크 컨퍼런스 발표 (런칭 이벤트 겸용)
- 보도자료: 인지과학 기반 난이도 측정의 의미 강조
- 성과 메트릭: 파트너사 10곳 계약, 일간 API 호출 10,000건

### Phase 3: 확장 (런칭 후 3~6개월)

- 곱셈/나눗셈 영역 확장 공지 및 베타 오픈
- B2B2C 채널(학원/학교) 본격 진입
- 성과 사례집 발간 (학습 효과 데이터 기반)
- 해외(동아시아) 시장 조사 및 현지화 검토

### Phase 4: 성숙 (런칭 후 6~12개월)

- 전 연산 영역 커버리지 완성
- B2C 직접 서비스 런칭 검토
- 교육과정 공식 연계 (교육부/교육청 협력)
- 성과 메트릭: MAU 100,000, 파트너사 50곳

## 5. 차별화 포인트

### vs. 기존 적응형 학습 플랫폼

| 차별화 요소 | 기존 솔루션 | 본 제품 |
|---|---|---|
| 난이도 산출 방식 | 정답률 기반 사후 분류 | 인지 부하 가중치 기반 사전 분류 |
| 투명성 | 블랙박스 알고리즘 | 6가지 가중치 공개 공식 |
| 세분화 수준 | 3~5단계 | W 23단계 + C 15레벨 |
| 문제 커버리지 | 샘플링 기반 | 15,000개 전수 채점 |
| 도입 즉시성 | 데이터 축적 필요 | 즉시 사용 가능 (사전 채점 완료) |

### 핵심 차별화 메시지 3가지

1. **"사전 채점 vs 사후 분류"**: 학생 오답 데이터 없이도 문제의 본질적 난이도를 즉시 판별한다. 콜드스타트 문제가 없다.

2. **"23단계 정밀도"**: 기존 상/중/하 분류 대비 약 8배 세밀한 난이도 구분으로, "약간 쉬움"과 "적정 도전"의 미세한 차이를 포착한다. [CLM-005, CLM-007]

3. **"화이트박스 AI"**: 왜 이 문제가 W=18인지 가중치별 점수로 완전히 설명 가능하다. 교사가 납득하고, 학부모가 신뢰하는 투명한 체계이다. [CLM-008]
