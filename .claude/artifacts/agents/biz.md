## 문제 정의 및 기회

### 핵심 문제

난산증(Dyscalculia)은 전 세계 학령기 아동의 3~6%에 영향을 미치는 신경발달적 수학 학습장애이다. 난독증(Dyslexia)과 유사한 유병률을 보이지만, 사회적 인지도와 전문 개입 도구 모두 현저히 부족하다.

기존 수학 학습 도구의 한계는 명확하다:

| 기존 접근 방식 | 한계 |
|--------------|------|
| 문제 반복 풀이 | "왜 틀렸는가"를 설명하지 못함 |
| 난이도 단순 조절 | 인지적 원인 분석 없이 양적 조절만 수행 |
| 범용 적응형 학습 | 난산증 특화 진단 체계 부재 |
| 오프라인 치료 | 접근성, 비용, 전문가 수 제한 |

Maththera는 PRD 템플릿의 Problem & Hypothesis 구조에 따라 '수감각 결손 -> 수학능력 저하 -> 난산증 고착'의 인과 체인을 정의하고, 'AI 진단-개입으로 수감각 회복 가능'이라는 가설을 검증한다. [CLM-010]

### 시장 기회

- 글로벌 에듀테크 시장: 2025년 1,891억 달러 -> 2034년 5,887억 달러 전망
- AI 교육 시장: 2025년 70.5억 달러 -> 2034년 1,123억 달러 급성장
- 한국 DTx 시장: 2023년 첫 승인 이후 급성장 중이나 난산증 특화 제품 부재
- 선점 기회: 85개 오답원인 MECE 분류와 뇌과학 기반 진단 체계를 갖춘 제품은 현재 시장에 없음

### Maththera의 해결 방식

Master Matrix는 '문제 정의 -> 오답 진단 -> 훈련 처방' 파이프라인의 상관관계를 유스케이스 시나리오로 정의한다. Cogassist Agent가 기획자가 명시하지 않은 복합적 오답 패턴에 대해서도 매트릭스 체계에 따라 자율적으로 정확한 매핑을 수행할 수 있게 하는 것이 핵심이다. [CLM-001]

### 보이스 톤 전략 (아동 중심 UX)

Maththera의 AI 피드백은 '틀려도 괜찮다'는 느낌으로 재시도를 유도하는 보이스 톤 가이드를 적용한다. 따뜻하고 격려하는 친구 같은 튜터 페르소나로, 비난/재촉/비교를 금지하며 반말이지만 부드러운 톤을 유지한다. [CLM-006]

기본 규칙은 전체 3문장 이내, 문장당 10자 이내(수식 포함 시 15자), 초저학년 기준 쉬운 단어만 사용, 한 번에 힌트 1개만 제시이다. 이 규격화된 UX 규칙은 AI 피드백의 품질 일관성을 보장하여 치료적 효과의 재현성을 높인다. [CLM-007]

상황별 응답 템플릿은 4가지 상황(정답/오답/과정상 실수/무응답)에 대해 구조화된 응답 패턴을 제공한다: [CLM-008]
- **정답**: 칭찬 + 확인/강화 ("좋아! 정답이야! 다음 가자!")
- **오답**: 공감 + 힌트 + 재도전 ("괜찮아. 하나만 더! 다시 해볼까?")
- **과정상 실수**: 칭찬 + 포인트힌트 + 재시도 ("거의 다 왔어! 자리만 봐! 한 번 더!")
- **무응답**: 안심 + 작은단계 + 선택지 ("괜찮아. 하나씩 하자. 어느 게 맞아?")

힌트 작성의 핵심 원칙은 '정답을 직접 말하지 않기'와 '아이가 할 수 있는 작은 행동으로 만들기'이다. 이는 ZPD 기반 scaffolding과 일치하여 학습 효과의 근거 기반 설계를 뒷받침한다. [CLM-009]

## 경쟁 환경 분석

### 유스케이스 기반 차별화 분석

Maththera의 4개 유스케이스 시나리오는 경쟁사가 다루지 못하는 진단적 깊이를 구체적으로 보여준다:

**유스케이스 1: 10의 보수 전략 미숙** [CLM-002]
- 정답을 맞혀도 RT 12초(느림)인 경우 OBS_OVERRELIANCE_COUNTING 감지
- T12(10짝찾기) -> T17(10만들어더하기)로 전략 전환 훈련 처방
- 차별화: "정답이어도 느리면 자동화 미완성"이라는 판단은 기존 에듀테크에 없음

**유스케이스 2: 자릿값 혼동** [CLM-003]
- 오답 6이 '4를 십의 자리 2에 더한 결과'와 일치함을 패턴 인식으로 감지
- 단순 연산 오류가 아닌 SPATIAL 인지 도메인 문제로 진단
- 차별화: 동일 오답이라도 인지적 원인을 구분하여 맞춤 훈련 제공

**유스케이스 3: 작업기억 부족/충동적 반응** [CLM-004]
- RT 0.8초(극도로 빠름) + OBS_RANDOM_GUESSING -> ATTENTION_L0_IMPULSE
- 수학 지식이 아닌 ATTENTION 도메인 개입 우선
- 차별화: 난산증과 ADHD 공존 장애 감별 가능 (DTx 근거)

**유스케이스 4: 변인 통제 Probe 기반 감별 진단** [CLM-005]
- 받아올림 변인(C3->C1)을 제거한 확인 문제를 자동 출제
- Case A(Probe 오답): 공간 결손 확정 -> T26(자릿값이해)
- Case B(Probe 정답): 절차 누락 확정 -> T27(받아올림개념)
- 차별화: 임상 수준의 감별 진단을 자동화

### 경쟁 구도

| 제품 | 주요 특징 | Maththera 대비 한계 |
|------|----------|-------------------|
| Dybuster Calcularis | ITS 기반 적응형, 30~45% 개선 보고 | MECE 오답원인 분류 부재, 전략 진단 불가 |
| DreamBox | 적응형 수학 플랫폼 | 난산증 비특화, 인지 도메인 진단 부재 |
| Khan Academy | 무료 수학 콘텐츠 | 진단/적응 기능 부재 |
| Todo Math | 유아/초등 수학 게임 | 난산증 비특화, 오답 원인 분석 부재 |

### MOAT (경쟁 우위)

1. **데이터 자산**: 15,050개 레벨 태깅 문제, 85개 오답원인 매핑, 유스케이스 시나리오
2. **진단적 깊이**: 4개 유스케이스로 검증된 전략 미숙/공간 혼동/주의력/감별 진단
3. **제품 표준**: 규격화된 보이스 톤 가이드로 AI 피드백 품질 일관성 확보
4. **네트워크 효과**: 사용 데이터 축적에 따른 진단 정확도 자동 향상
