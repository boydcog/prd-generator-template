# 비즈니스/전략 분석: 수학 문제 레벨링 품질 검증

## 1. 문제 정의 및 기회

### 핵심 문제
에듀테크 분야에서 학생 수준에 맞는 수학 문제를 자동 추천하려면 객관적이고 일관된 난이도 기준이 필수적이다. 현재 대부분의 수학 학습 플랫폼은 교사의 주관적 판단이나 단순 정답률에 의존하여 문제 난이도를 결정하고 있으며, 이는 학습자 개인별 최적 난이도 구간(ZPD, Zone of Proximal Development)을 정밀하게 타겟팅하지 못하는 결과를 초래한다.

### 기회
본 프로젝트가 분석하는 W1~W6 가중치 체계는 수학 문제의 인지적 난이도를 6개 차원으로 분해하여 정량화한다. 데이터셋은 약 15,050개 덧셈/뺄셈 문제를 W(1~23) 23단계 미세 레벨과 C(1~15) 15등급 교육과정 등급으로 이중 매핑하고 있다 [CLM-001, CLM-006]. 이 체계가 검증되면:

- **적응형 학습 엔진**의 핵심 난이도 모듈로 활용 가능
- **교사 대시보드**에서 학급별/학생별 난이도 분포 시각화 가능
- **문제 은행 자동 분류** 시스템의 기반 알고리즘으로 확장 가능

## 2. 대상 사용자/고객

| 사용자 유형 | 니즈 | 가치 제안 |
|------------|------|----------|
| **에듀테크 개발팀** (1차) | 문제 난이도 자동 분류 API | 규칙 기반 레벨링으로 설명 가능한 추천 시스템 구축 |
| **수학 교사** (2차) | 학생 수준에 맞는 문제 배정 | 23단계 미세 레벨로 정밀한 학습 경로 설계 |
| **학습 콘텐츠 제작자** (3차) | 신규 문제의 난이도 사전 예측 | 가중치 공식에 따라 문제 생성 단계에서 레벨 확정 |
| **학부모/학생** (최종) | 적절한 난이도의 학습 경험 | 좌절/지루함 없는 최적 도전 구간 유지 |

## 3. 비즈니스 목표 및 성공 지표 (KPI)

### 비즈니스 목표
W1~W6 가중치 체계가 실제 학습 난이도를 정확히 반영하는지 검증하여, 에듀테크 제품의 적응형 학습 엔진에 통합할 수 있는 신뢰할 수 있는 난이도 분류 체계를 확립한다.

### 핵심 KPI

| KPI | 목표값 | 측정 방법 |
|-----|--------|----------|
| **W_총점-실제 정답률 상관계수** | r >= 0.75 | 파일럿 학습자 데이터와 교차 검증 |
| **C등급 간 정답률 단조감소 비율** | >= 90% | C(n)등급의 평균 정답률이 C(n+1)보다 높은 쌍의 비율 |
| **레벨 건너뛰기 비율** | <= 5% | 학습자가 2단계 이상 건너뛰는 비율 |
| **W-C 매핑 일관성** | >= 95% | 동일 W_총점에서 C등급이 일관된 비율 |
| **가중치 체계 커버리지** | 100% | 1~2자리 덧셈/뺄셈 전 조합 커버 |

## 4. 경쟁 환경 분석

### 현재 시장 상황

에듀테크 수학 학습 시장은 AI 기반 적응형 학습이 주류를 이루고 있으나, 대부분 블랙박스 모델에 의존한다.

| 경쟁자/대안 | 난이도 결정 방식 | 강점 | 약점 |
|------------|----------------|------|------|
| **매쓰플랫** | AI 기반 정답률 분석 | 실시간 적응 | 난이도 기준 불투명 |
| **클래스팅 AI** | 학습 이력 기반 ML | 대규모 학습 데이터 | 콜드 스타트 문제 |
| **칸아카데미** | 수동 태깅 + 정답률 | 교육 전문가 검증 | 확장 비용 높음 |
| **토도수학** | 게이미피케이션 기반 | 학습 동기 부여 | 정밀 레벨링 미흡 |
| **본 프로젝트** | 규칙 기반 6차원 가중치 | 투명성, 설명 가능성 | 실증 검증 미완료 |

### 차별화 포인트

1. **설명 가능성**: W1~W6 각 차원의 점수가 왜 그렇게 매겨졌는지 교사/학부모에게 투명하게 설명 가능 [CLM-002, CLM-003]
2. **심리적 요인 반영**: W5 특수 숫자 감점은 학습자의 인지 심리학적 특성(10의 배수 안착점, 규칙성 인지)을 반영한 고유 요소 [CLM-005]
3. **미세 레벨링**: 23단계 W레벨은 기존 5~10단계 분류 대비 3~4배 정밀한 난이도 조정 가능 [CLM-006]
4. **콜드 스타트 무관**: 학습 이력 데이터 없이도 문제 속성만으로 즉시 레벨 부여 가능

### 수익 모델 (B2B SaaS 중심)

| 수익원 | 모델 | 대상 |
|--------|------|------|

| **문제 은행 라이선스** | 연간 구독 | 학원/학교 |

| **학습 분석 대시보드** | 사용자당 월정액 | 교사/학원 |

### 비용 구조

| 항목 | 초기 비용 | 운영 비용 | 비고 |
|------|----------|----------|------|
| **가중치 검증 연구** | 높음 | - | 파일럿 테스트, 교육 전문가 검토 |
| **데이터셋 확장** | 중간 | 낮음 | 곱셈/나누기, 3자리 이상 확장 |
| **API 인프라** | 낮음 | 낮음 | 규칙 기반이므로 연산 비용 최소 |
| **콘텐츠 유지보수** | - | 중간 | 교육과정 변경 시 C등급 매핑 업데이트 |

### 비용 우위
규칙 기반 체계는 AI/ML 모델 대비 학습 데이터 수집, GPU 인프라, 모델 재학습 비용이 불필요하여 운영 비용이 현저히 낮다. 약 15,000개 문제의 레벨이 사전 계산되어 있으므로 런타임 비용은 단순 조회 수준이다.

## 6. 핵심 인사이트

### 가중치 기여도 분석

데이터에서 확인된 각 W차원의 최대 점수 기여분:

| 차원 | 최대 기여 | 전체 대비 비중 | 비고 |
|------|----------|--------------|------|
| W1 연산 종류 | 1.5점 | ~14% | 덧셈 vs 뺄셈 이진 분류 [CLM-002] |
| W2 자릿수 | 4.0점 | ~36% | 2자리x2자리 비대칭 시 최대 [CLM-004] |
| W3 숫자 크기 | 1.5점 | ~14% | 6~9 포함 시 최대 [CLM-008] |
| W4 복잡성 | 5.0점 | ~45% | 연속 carry/borrow 시 최대 [CLM-003] |
| W5 특수 숫자 | -1.5점 | 감점 | 10의 배수 + 동일 숫자 [CLM-005] |
| W6 표현 방식 | 0.5점 | ~5% | 현재 일률 적용 |

W4(절차적 복잡성)가 전체 난이도 분산의 가장 큰 동인이며, W2(자릿수)가 그 다음으로 영향력이 크다 [CLM-003, CLM-007]. 이 두 차원만으로 전체 레벨 변동의 약 80%를 설명할 수 있다.

### W_총점과 C등급 매핑의 비선형성

동일한 W_총점(7.5)에서도 덧셈 문제는 C(8), 뺄셈 문제는 C(9)로 배정되는 사례가 확인되었다 [CLM-009]. 이는 C등급 매핑에 W_총점 외 추가 규칙(연산 방향 보정 등)이 적용되고 있음을 시사하며, 이 규칙의 투명한 문서화가 시스템 신뢰도의 핵심 과제이다.

## 8. 미해결 질문

1. W_총점에서 C등급으로의 매핑 함수가 비선형적인데, 이 매핑 규칙의 명시적 기준이 필요하다 [OQ-001]
2. 곱셈/나누기 확장 시 기존 W 프레임워크 내 확장이 가능한지 검토 필요 [OQ-002]
3. W6 표현 방식 차원이 현재 일률 0.5점인데, 문장제 등 변형 적용 계획 확인 필요 [OQ-003]
4. 실제 학습자 체감 난이도 데이터와의 교차 검증 여부 확인 필요 [OQ-004]
