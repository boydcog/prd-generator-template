## 1. 사용자 인사이트 요약

### 대상 사용자: 난산증(Dyscalculia) 아동

난산증은 지능이나 교육 기회의 부족이 아니라, 후두정엽(수 표상)과 전전두엽(집행 제어) 사이의 신경학적 연결성 결함에서 기인하는 발달적 장애이다. 이는 단순한 '수학 포기'나 학습 부족이 아닌 뇌 회로 수준의 구조적 문제로, 표적화된 인지 개입이 필요함을 시사한다.
[출처: dyscalculia-cse-cognitive-model/chunk-0000]

### 핵심 인사이트

1. **C-S-E 3축 인지 모델이 진단의 근간이다**: 개념(C: 장기기억의 수 표상, C1~C6), 전략(S: 전전두엽의 문제 해결 설계, S1~S7), 집행(E: 작업기억의 처리 부하, E1~E3)의 3축으로 난산증의 인지 결함을 분리한다. 정서/환경 요인을 배제한 순수 인지 문제로 규정함으로써 진단 범위를 명확히 하되, 개념 축은 하위 단계(기수성)에서 상위 단계(가역성)로 누적적으로 구성되어 하위 결함이 상위 전체를 무너뜨리는 위계적 구조를 갖는다.
[출처: dyscalculia-cse-cognitive-model/chunk-0001, chunk-0002]

2. **정오답만으로는 진단이 불충분하다**: 정답을 맞혔더라도 전략 불일치(예: MAKE10 전략 필요 문제에서 COUNTING 사용)가 감지되면 유창성 훈련이 처방되어야 한다. 유스케이스 1에서 8+6=14 정답이지만 RT 12초 + OBS_OVERRELIANCE_COUNTING 행동 관찰로 전략 부재를 진단한 사례가 이를 입증한다. 반응 시간(RT)과 행동 관찰 태그의 결합이 정밀 진단의 필수 요소이다.
[출처: maththera-matrix-usecase/chunk-0000, UC1]

3. **작업기억 부하가 핵심 병목이다**: 난산증 아동은 작업기억(E축) 임계치가 낮아, 문제 난이도가 인지 레벨을 초과하면(L < C) 충동적 반응(찍기)이나 중간 단계 망각이 발생한다. 유스케이스 3에서 RT 0.8초의 극도로 빠른 반응은 수학적 지식 부족이 아닌 ATTENTION(주의력) 도메인의 문제로, 비인지적 개입이 우선된다.
[출처: dyscalculia-cse-cognitive-model/chunk-0004, maththera-matrix-usecase/chunk-0000 UC3]

4. **변인 통제 Probe가 정밀 진단의 차별화 요소이다**: 28+15=33 오답에서 정답(43)과의 -10 차이를 포착한 후, 받아올림 변인을 제거한 20+15 문제를 즉시 출제하여 공간 지각(SPATIAL) vs 절차 수행(STRATEGY) 원인을 분리하는 Probe 기반 진단이 Maththera의 핵심 차별화 로직이다.
[출처: maththera-matrix-usecase/chunk-0000, UC4]

5. **정서적 안전망이 인지 훈련의 전제 조건이다**: AI 피드백은 '따뜻하고 격려하는 친구 같은 튜터' 페르소나로 설계되어, 반말 사용 + 3문장 이내 + 10자 이내 규칙을 따른다. '틀려도 괜찮다'는 느낌으로 재시도를 유도하며, 비난/재촉/비교를 절대 금지하여 수학 불안(Math Anxiety)과 학습된 무력감을 경감시킨다.
[출처: prd-template-voice-tone-guide/chunk-0000]

## 2. 증거 맵 (출처별 핵심 발견)

### 출처 1: C-S-E 인지 모델 프레임워크 (dyscalculia-cse-cognitive-model)

| 영역 | 핵심 발견 | 교육학적 근거 | 신뢰도 |
|------|----------|-------------|--------|
| 뇌과학적 정의 | 후두정엽-전전두엽 연결성 결함이 난산증의 신경학적 원인 | 발달 신경과학 연구 합치 | high |
| 개념(C) | C1 기수성 ~ C6 가역성 6단계 위계. 하위 결함이 상위 전체를 무너뜨림 | Piaget의 수 개념 발달 단계 | high |
| 전략(S) | S1 전부세기 ~ S7 유형분류 7단계. 각 S는 대응하는 C에 1:1 의존 | Siegler의 겹치는 파도(Overlapping Waves) 모델 | high |
| 집행(E) | E1(정보유지량) x E2(절차단계) x E3(수크기) = 48단계 난이도 매트릭스 | Baddeley의 작업기억 모델 | high |
| 진단 시사점 | C축: 시각적 수 표상 훈련 / S축: 메타인지 발문 훈련 / E축: 단계적 연산 훈련 | 각 축별 차별화된 개입 | high |
| 시각 메타포 | 기찻길-블록 통합 메타포로 C1~C6/S1~S6을 일관된 스토리로 연결 | 체화된 인지(Embodied Cognition) | medium |

## 4. 미지 영역

### 고우선순위

1. **연산 영역 확장**: 현재 C-S-E 모델과 48단계 매트릭스는 덧셈에만 특화되어 있다. C6(가역성)이 뺄셈의 기초라는 언급은 있으나, 뺄셈/곱셈/나눗셈 영역의 C-S-E 매핑과 난이도 매트릭스 확장 설계가 필요하다.

2. **다중 원인 오답 처리**: UC4(오답 원인 중복 책정)에서 Probe 기반 이진 분기(Case A vs Case B)는 설계되었으나, 두 원인이 동시에 존재하는 경우나 Probe 결과가 애매한 경우(예: 느린 RT로 정답)의 판정 기준이 없다.

### 중우선순위

3. **정서-인지 통합 모델**: C-S-E가 순수 인지 모델로 정서 요인을 배제했지만, 수학 불안(Math Anxiety)이 E축(작업기억 부하)에 직접 영향을 미친다는 연구가 다수 존재한다. 정서 상태에 따른 진단/처방 분기 로직이 필요하다.

4. **Agent 자율 매핑 검증**: 4개 유스케이스를 Cogassist Agent에 학습시켜 복합 오답 패턴에 자율 매핑할 수 있다는 가설은 실증 데이터 없이 기술 단계에 있다. 파일럿 테스트로 실제 아동 데이터 기반 매핑 정확도를 측정해야 한다.

### 저우선순위

5. **난이도 매트릭스 상위 구간 중복**: 48단계 중 L22~L48 구간에서 유사한 패턴(예: L22-24와 L37-39)이 관찰된다. 이 구간의 차별화 기준을 정밀하게 검증하고 중복 항목을 정리할 필요가 있다.

6. **시각 메타포 효과 검증**: 기찻길-블록 통합 메타포의 교육적 효과가 실증적으로 검증되지 않았다. 체화된 인지 접근이 난산증 아동에게 전통적 교구 대비 유의미한 효과를 보이는지 A/B 테스트가 필요하다.
