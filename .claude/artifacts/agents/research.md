# Research Analysis: 수학 문제 레벨링 품질 검증

## 사용자 인사이트 요약

본 분석은 수학 난이도 채점 기준표(15,050줄, 151청크)를 체계적으로 검토하여 W1~W6 가중치 체계가 실제 학습 난이도를 정확히 반영하는지 검증한다.

### 핵심 발견

1. **가중치 체계의 구조적 편중**: W4(절차적 복잡성)가 총점의 최대 45%를 단독 결정하며, 나머지 5개 차원의 영향력이 상대적으로 희석되어 있다.
2. **C단계 매핑의 비결정론성**: 동일 W총점/W단계를 가진 문제도 C단계가 다르게 배정되는 사례가 다수 발견되며, 추가 매핑 규칙이 문서화되어 있지 않다.
3. **W6 차원의 무실화**: 표현 방식(W6)이 모든 문제에서 0.5점으로 고정 적용되어 실질적 변별 기능이 없다.
4. **실증 데이터 부재**: 전문가 판단 기반 구성으로 보이며, 학생 정답률/풀이 시간 등 실증적 검증 데이터가 부재하다.

## 미지 영역 (Unknowns)

### 높은 우선순위

1. **W총점 → W단계 매핑 공식**: 데이터에서 W총점 0→W1, 0.5→W2, 1→W3, 1.5→W4, 2→W5, 2.5→W6, 3→W7 등으로 보이나 정확한 매핑 테이블이 문서화되어 있지 않음
2. **W단계 → C단계 매핑 규칙**: 비선형적이고, 0 포함 여부, +0 형태 문제 등 추가 조건이 C단계에 영향을 미침. 규칙의 완전한 목록 부재
3. **가중치 결정 근거**: 각 가중치 값(W1=1.5, W4=3/5 등)의 산출 근거가 전문가 합의인지 데이터 기반인지 불명

### 중간 우선순위

4. **W3 캡 효과**: 6~9 포함 최대 1.5점 제한이 3자리 수에서도 동일하게 적용되는지
5. **W6 확장 계획**: 문장제, 가로셈 등 표현 방식별 점수 체계 미정의
6. **곱셈/나눗셈 확장**: W1~W6 프레임워크가 곱셈/나눗셈에 어떻게 적용될지 미정의
7. **학년/나이별 차이**: 같은 가중치 체계가 모든 학년에 동일하게 적용되는지, 발달 단계별 조정이 필요한지
